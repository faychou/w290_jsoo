2、封装
2.1 理解
隐藏对象内部实现的细节，对外提供一致的访问接口。

2.2 怎么封装
  >> 将定义类、函数、插件等功能的代码进行打包，便于维护和管理。

  >> 对类的成员属性和方法进行访问权限的控制（读写）。
    * private：私有
    * public：公共
    * protected：受保护的，只能通过特定的接口来访问

    在 JS 中并没有对这三个特性提供语法支持，只能通过函数的语法特性来模拟。

2.3 封装的特征
  * 将不需要公开的数据设置为私有；
  * 使用特权方法对部分数据进行权限控制（setter/getter）；
  * 可以对外扩展，对内应该关闭修改。

3、多态
3.1 理解
方法可以根据传递的参数类别、个数等进行区别，返回不同的结果。

3.2 案例
$().css('color'); // 获取颜色
$().css('color', 'red'); // 设置颜色
$().css({
  color: 'red',
  width: '100px'
}); // 同时设置多个属性

4、继承（重难点）
4.1 理解
子类拥有父类所有的属性和方法，从而避免去做相同的事情。

4.2 怎么实现继承
  * 子类实例化的对象必须具备父类所有的属性和方法；
  * 子类实例化的对象应该属于父类；
